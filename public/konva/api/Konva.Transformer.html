Class: Transformer

Konva. Transformer

new Transformer(config)

Transformer constructor. Transformer is a special type of group that allow you transform Konva primitives and shapes.
Parameters:
Name Type Description
config Object
Properties
Name Type Argument Description
resizeEnabled Boolean <optional>
 Default is true
rotateEnabled Boolean <optional>
 Default is true
rotationSnaps Array <optional>
 Array of angles for rotation snaps. Default is []
rotateHandlerOffset Number <optional>
 Default is 50
padding Number <optional>
 Default is 0
lineEnabled Number <optional>
 Should we draw border? Default is true
keepRatio Boolean <optional>
 Should we keep ratio when we are moving edges? Default is true
enabledHandlers Array <optional>
 Array of names of enabled handles
Example
var transformer = new Konva.Transformer({
  node: rectangle,
  rotateHandlerOffset: 60,
  enabledHandlers: ['top-left', 'top-right', 'bottom-left', 'bottom-right']
});
layer.add(transformer);
